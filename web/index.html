PowerLinux

The best linux distro for PowerPC!

PowerLinux is currently in VERY early beta, and may be unstable, use at your own risk.

PowerLinux is based on Debian and is designed to be the best linux distribution for old PowerPC computers. (e.g. macs from before 2006)

How do I know if my computer uses PowerPC?

The most common PowerPC computers are pre-2006 macs, you can find out if your mac is using PowerPC by checking if you are using
Mac OS 10.5 or earlier, as PowerPC cannot run Mac OS 10.6 or later.
